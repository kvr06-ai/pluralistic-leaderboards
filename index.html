Headline metrics — max γ̂ over all prefix sizes k

A committee is locally stable when γ̂ ≤ 1. Lower max γ̂ = aggregation rule respects minority sub-populations more reliably. Algorithm 3 should dominate Bradley-Terry at low φ (the regime where pluralism matters most).

Stability curves — γ̂ vs prefix size k

Each line plots γ̂ for prefix sizes k = 1 through 19. The horizontal threshold at γ̂ = 1 is the local-stability boundary; algorithms staying below it preserve representation across all top-k committees.

Top-20 rankings — side-by-side comparison

Rank-change badges relative to Bradley-Terry baseline. Green = model moved up vs BT. Orange = model moved down vs BT.

Category mixture — prompt distribution & per-category top model

12 prompt categories from a 4-base × 3-difficulty flattening of LMArena's category_tag schema. Mixture weights set by per-category battle count. Top model is the per-category Bradley-Terry winner.
